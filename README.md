Modernize Free Next.js 14 Admin Template with Material Ui + Typescript 

# Installation 👨🏻‍💻

> We recommend you use npm

1. Install all packages

```
npm i
```

2. Run Development Server

```
npm run dev
```

3. Build your project

```
npm run build
```
